Welcome to Rental Buddy! Below are the software requirements for running this system.
python 3.0+
pip
flask
pandas
requests
time
numpy
tagui

Please open terminal from this folder and follow the following steps:
1) Type pip install -r requirements.txt
2) If you encounter any error in using Tagui, Please refer 'https://www.yuchenkuang.com/category/TagUI/TagUI-Python-Setup/' for installation steps.
